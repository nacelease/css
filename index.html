Main

starting an html page always start with the main tag and add 'role' inside the opening of the main tag. It should look like this: main role="main". All of it should be within <> as well as it ends with .

Section

The section tag divides the content into section and subsections. The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

Heading HTML

HTML defines six levels of headings. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the biggest and H6 the smalllest.
